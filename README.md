# GoldenEye Project

## What you need

-   JDK 1.8+
-   Maven 3+

## What it's included into the project

- Spring Boot
- FreeMarker

## Run de application

    mvn spring-boot:run

## How to modify the final string

    To modify the final string, you just need to implement the new code to the method getEmail() in the class User.
